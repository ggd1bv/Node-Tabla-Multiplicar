# NOTAS:

Primer App de Node.

## Ayuda:
    --help         Show help                [boolean]
    --version      Show version number      [boolean]
    -b, --base     Base de la tabla         [number] [required] 
    -l, --listar   Muestra la tabla         [boolean] [default: false]
    -h, --hasta    Cantidad a multiplicar.  [number] [default: 10]
        
## Instalación
```
npm i
```

## Comandos útiles
```
node app -b 9 -l  -h 15
node app -b 12 -l
node app -b 18 
```
